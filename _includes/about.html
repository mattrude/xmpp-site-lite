This Service

This server provides instant messaging service, via the XMPP protocol, to users with an account on this domain. An XMPP server provides basic messaging, presence, and XML routing features.

This site is running prosody version 0.10 on Ubuntu version 16.04 LTS.. The source for prosody may be found at hg.prosody.im. The program prosody is licensed under the MIT/X11 license.
